## Description
This is a personal learning project inspired by the classic Snake game.  
The goal is to practice front-end development, JavaScript logic, and team collaboration by recreating the core mechanics: player movement, collision detection, scoring, and game-over conditions.  

This project was done in a learning context and is not intended to be a commercial version.

## Technologies
- HTML5
- CSS3
- JavaScript
- Canvas
- Tailwind CSS (via CDN)

## Features
- Snake movement with arrow keys
- Growth of the snake when collecting food
- Collision detection
- Score tracking
- Game Over screen
- Restart functionality

## Project Management
This project was developed as part of a learning process, following structured project management practices:

1. **Requirement Analysis**  
   - Defined the game mechanics, objectives, and main features.
   - Identified user stories for gameplay, scoring, and UI.

2. **Planning**  
   - Created a Gantt chart to organize tasks and deadlines.
   - Broke down the project into **User Stories** (US) and tickets using **Jira**.

3. **Team Development**  
   - Tasks were assigned among team members.
   - Development was done collaboratively with regular updates.
   - Small code reviews were made to mimic a larger project.

4. **Tools Used**  
   - **Jira**: backlog, US, task tracking, sprints
   - **GitHub**: version control
   - **VS Code**: development environment

## Purpose
- Practice JavaScript and Canvas logic
- Work on front-end layout and UI design
- Learn collaborative development and project management
- Improve problem-solving through a fun game project
